Моя технологія та обладнання

FDM (Fused Deposition Modeling)

Це технологія пошарового наплавлення пластику. Ідеально підходить для створення міцних прототипів, корпусів, функціональних деталей, кріплень та багато іншого.

Моє обладнання:

Creality Ender 3 V3 KE — сучасний принтер, що забезпечує:

  • Область друку: 220 x 220 x 240 мм
  • Високу швидкість: до 500 мм/с
  • Точність: до 0.1 мм
  • Роботу з різними матеріалами: PLA, PETG, ABS, TPU.

Також виконую постобробку деталей (видалення підтримок, шліфування) за допомогою технічного фену для досягнення кращого вигляду.

Матеріали для друку

Я використовую лише якісні та перевірені матеріали для досягнення найкращого результату.

PLA (Полілактид)

Біорозкладний та простий у друці пластик. Ідеальний для декоративних моделей, прототипів, іграшок та виробів, що не зазнають високих навантажень і температур.

  • Висока деталізація
  • Екологічність
  • Низька термостійкість

PETG (Поліетилентерефталат-гліколь)

Міцний, гнучкий та стійкий до хімічних речовин матеріал. Чудово підходить для функціональних деталей, механічних частин, контейнерів та захисних екранів.

  • Висока міцність
  • Вологостійкість
  • Схильний до подряпин

ABS (Акрилонітрилбутадієнстирол)

Термостійкий та ударостійкий пластик, який використовується в промисловості (наприклад, кубики LEGO). Підходить для корпусів електроніки та деталей автомобілів.

  • Стійкість до температур
  • Легко обробляється
  • Потребує спец. умов друку

TPU (Термопластичний поліуретан)

Гнучкий, еластичний матеріал, схожий на гуму. Використовується для створення чохлів для телефонів, прокладок, гнучких з'єднань та іграшок.

  • Гнучкість та еластичність
  • Стійкість до зносу
  • Складний у друці

Що можна виготовити?

FDM-друк відкриває безліч можливостей для інженерів, майстрів та ентузіастів.

Функціональні прототипи

Перевірка ергономіки, дизайну та функціональності перед серійним виробництвом.

Деталі для ремонту

Заміна зламаних пластикових деталей в побутовій техніці, авто, меблях (шестерні, кріплення, ручки).

Кастомні вироби

Унікальні корпуси для електроніки (Raspberry Pi, Arduino), органайзери, підставки, елементи декору.
